vp-pager
========
>一个简单的分页组件

### 依赖 ###
1. vue.js


### 主要功能 ###


### props ###
* visibleCount：显示多少个格子 默认10
* showFirstBtn: 是否固定显示第一页 默认 true
* showLastBtn: 是否固定显示最后一页 默认 true
* showShortCut: 显示跳转到某一页 默认 false
* previous: 上一页的文案 默认 上一页
* next: 下一页的文案 默认 下一页
    注意：当previous ,next 的值为 '' 空字符串的时候下一页/下一页的文案分别为  <,> 并且长度会变短。</
